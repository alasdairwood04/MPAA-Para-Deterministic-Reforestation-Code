# MPAA Group Project: Optimising Reforestation in Pará

## Project Overview
This repository contains the deterministic mathematical programming models developed for Part II of the "Mathematical Programming in Advanced Analytics" (BUST10134) group project. Our project aims to optimize the allocation of a US$ 489,850,000 reforestation budget across 144 municipalities in Pará, Brazil, to maximize ecological returns (SDG 15) while mitigating reversal risks.
## Folder Structure
* **Data/**: *(Note: Excluded from this submission due to file size limits)*. Contains the primary dataset `MPAA_model_data.csv` incorporating deforestation urgency, biodiversity density, delivery costs, and reversal risk indices.
* **Deterministic Code/**: Contains the Jupyter Notebooks executing our Pyomo/Gurobipy models.
* **Outputs/**: Contains all CSV tables generated by the models (e.g., payoff tables, optimal allocations, sensitivity analysis data).
* **Figures/**: Contains generated visualizations, including radar plots and budget sensitivity graphs.
## Notebook Execution Order & Descriptions
The implementation of our Mixed-Integer Linear Programming (MILP) models is divided into sequential Jupyter notebooks to ensure clarity and modularity:

1. **`01_individual_objective_analysis.ipynb`**: Solves the MILP independently for each objective (Biodiversity/Urgency maximization vs. Reversal Risk minimization) to establish the theoretical upper and lower bounds (Utopia and Nadir points).
2. **`02_main_lexicographix_model.ipynb`**: Implements the core multi-objective optimization using a Lexicographic/Hierarchical approach, allowing us to prioritize objectives while applying epsilon-constraint logic to secondary goals.
3. **`03_heuristic_model.ipynb`**: A benchmark heuristic approach (e.g., greedy algorithm based on cost-effectiveness) to compare against our exact MILP solutions.
4. **`04_nadir_utopia_tolerance_sensitivity.ipynb`**: Generates the payoff table and analyzes the trade-offs between competing objectives using the Nadir/Utopia values.
5. **`05_adverse_investor_sensitivity.ipynb`**: Explores a scenario prioritizing minimum reversal risk, adapting the model weights to reflect a highly risk-averse funding entity.
6. **`06_budget_sensitivity_analysis.ipynb`**: Parametric analysis varying the primary constraint (the US$ 489.85M budget) to evaluate how marginal changes in funding impact the optimal spatial allocation and total hectares restored.
## How to Run
1. Ensure the `Data` folder is placed in the root directory.
2. Install dependencies: `pip install gurobipy pandas matplotlib seaborn numpy`.
3. An active Gurobi Academic License is required.
4. Run the notebooks sequentially from `01` to `06` to reproduce the Outputs and Figures.
